## Slack Bot

AWS Lambda Python 3.6.1 + API Gatewayで動作します。

### 環境変数

* DOCOMO_API_KEY : Docomo雑談APIのトークン
* LOG_LEVEL : ログレベル
* BOT_NAME : BOTの名前
